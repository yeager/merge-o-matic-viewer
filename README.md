# Merge-o-Matic Viewer

Merge helper for Ubuntu-Debian package synchronization.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install merge-o-matic-viewer
```
